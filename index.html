作品

グループ創作

「Petit Fleurs」Webサイト

職業訓練校にて、チームで制作した作品です。
別グループが担当したクライアント役より「幅広い世代の女性に親しまれる、花と癒しをテーマにした架空のフラワーショップの店舗サイト」の依頼を受け、トップページコンタクトページのデザインと有難うページのデザイン・コーディングを担当しました。

プロジェクト概要
  • 制作年：2025年
  • ページ数：6ページ
  • 制作期間：約2週間（デザインフェーズ：約12日）
  • チーム人数：3名
  • 使用ツール：Visual Studio Code、Canva
ターゲット層

20代〜60代のご夫婦・カップル、また花やアロマ、癒しを好む女性層。
幅広い年代に親しまれるよう、温かみのある写真を多く使用し、ビジュアルで魅せる構成に。

コーポレートカラー / デザイン方針

ブランドカラーはグリーンを基調に、全体的に淡いグラデーションでやさしさと安心感を演出。
フォント・余白の使い方にも配慮し、ナチュラルで落ち着いた雰囲気を表現。

サイトの目的

問い合わせやワークショップ予約数の向上
ユーザーのアクションを促すため、各所に「お問い合わせ」やSNSリンクを挿入し、動線を意識した設計に。

担当領域（私の役割）
  • トップページのHTML/CSSコーディング
  • レスポンシブ対応（スマホ/タブレット）
  • アニメーションの研究と実装
  • チームメンバーとのコミュニケーション、進捗共有

スライドショーの工夫

目的は「誰でも見やすく、印象に残る、また来たくなる」為に、スライドショー機能を実装し、ヒーローイメージを作成しました。
スライドショーは、ユーザーが画像をクリックすることで次の画像に切り替わるように設計されています。

特徴・工夫ポイント
  • スライドサイズ：画面幅いっぱいの大きな画像を表示 → 一目で店の雰囲気や魅力が伝わる
  • ナビゲーションボタン：初期状態は半透明に設定して「視線を邪魔しない」デザイン。
    → ホバー時には緑に発光（hoverで目立つ） → 次回訪問時も印象に残る
  • スマホ対応：指でも簡単に操作できる配置とサイズ
  • テキスト最小化：画像の世界観を邪魔せず、シンプルな構成
このデザインが与える効果
  • サイト訪問者の第一印象がアップ
  • 写真中心の構成で「記憶に残りやすい」
  • 高齢層にもわかりやすく、若者にも「映える」印象
  • 次のアクション（スクロールやボタン操作）を自然と誘導する

イベント表示の工夫

イベント開催日をホバーすると、
🕒「あと何日でイベントが始まるか」を表示するカウントダウンが出現します。

イベントの状況は色分けで直感的に把握できます：
  • 🟢 準備中（開催まで10日以上）：落ち着いた印象
  • 🟡 もうすぐ開催（10日以内）：注意喚起として視認性UP
  • 🔴 開催中：緊急性・注目度を強調
  • ⚪ 終了済み：柔らかく情報整理
このデザインが与える効果
  • ユーザーは「今どういう状態なのか」を一目で理解でき、
  • ページを読む負担が軽減され、快適でスマートな体験が実現します。

CONTACTページの工夫

お問い合わせ後のサンクス画面では、
✔️「丁寧で温かみのあるメッセージ」と
✔️「花のイメージ画像」が組み合わされ、
信頼感・安心感を与える設計になっています。

  • メッセージカード風のデザインで、
    実店舗の“人の温もり”を感じさせる工夫がされています。
  • 背景には実際の花の写真を使うことで、
    ブランドイメージと統一感を保ちつつ、
    視覚的な癒し効果を与えています。
アニメーションの工夫
  • PC表示では、
    カードがふわっと立体的に回転しながら表示される3Dアニメーションで、
    高級感・印象的な演出を実現。
  • スマホ表示では、
    画面上からスッとスライドインする動きを取り入れ、
    テンポよく軽やかなユーザー体験を提供。
このデザインが与える効果
  • 問い合わせ後の不安を和らげる効果
  • また利用したい」と感じる好印象
    • をユーザーに与えることができると思って設定しました。

個人創作

「ベトナムの味仙へようこそ❣」Webサイト

台湾と日本の味を融合した60年以上の伝統を誇る「味仙」は、現在ホーチミン市で店舗を展開中。
ベトナムのユーザーに向け、独自の辛さと旨みを伝えるためのブランドサイトを制作しました。

プロジェクト概要
  • 制作年：2025年
  • ページ数：5ページ（現在一部制作中）
  • 制作期間：約1週間
  • チーム人数：1名（個人制作）
  • 使用ツール：Figma、Visual Studio Code、Canva, ChatGPT
ターゲット層

幅広い年齢層を対象に、友人同士・家族・会社の飲み会など多様なニーズに対応（辛さが苦手な人もOK）。

サイトの目的

Misenの存在を広く知ってもらい、来店のの増加を目指すブランディング施策の一環として企画しました。

デザイン方針

鮮やかな赤と黄色を基調にし、食欲をそそる写真を効果的に配置。辛さと温かみの両方を感じられる配色とトーンで親しみやすさを演出しました。

担当領域（私の役割）
  • 全体設計・情報構造の整理
  • デザイン作成およびコーディング（HTML/CSS/JS/JQ）
  • レスポンシブ対応
  • ブランドに合ったトーン設計

ホーム紹介ページ

当ホームページは、視覚的なインパクトとユーザビリティを重視したデザインアプローチにより、国内外の利用者が快適に閲覧・操作できるよう工夫されています。

  • ライト／ダークテーマの切り替えが可能
  • 高齢者にも配慮したシンプルかつ見やすいレイアウト
画像とインタラクションの工夫

  • 感情に訴えかける設計でUXを向上
まとめ

  • ユーザー体験を第一に考えた設計思想
  • エンゲージメントとリピーター獲得を強化

メニューページ紹介

この「メニュー」ページは、ベトナム人から海外のお客様まで誰でもスムーズ・簡単・直感的に注文できるように設計されています。

料理カテゴリの明確な分類
  • ジャンル "中華料理"「台湾料理」メイン料理：麵類ビリカラ料理、ニンニク料理＆ご飯物
